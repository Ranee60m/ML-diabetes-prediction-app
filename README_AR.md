# 🩺 تطبيق التنبؤ بمرض السكري

تطبيق ويب يعتمد على تعلم الآلة للتنبؤ بخطر الإصابة بمرض السكري بناءً على بيانات المريض السريرية، مبني بـ **Streamlit** ومدرَّب على **مجموعة بيانات PIMA Indians للسكري**.

---

## 📌 نظرة عامة

يقارن هذا المشروع أربعة نماذج للتعلم الآلي للتنبؤ بما إذا كان المريض في خطر الإصابة بالسكري، مع تقديم تنبؤات فورية عبر واجهة ويب تفاعلية.

| النموذج | دقة التدريب |
|---|---|
| 🌲 شجرة القرار | 79.84% |
| 🌳 الغابة العشوائية | 84.27% |
| 📉 الانحدار اللوجستي | 71.14% |
| ⚡ Gradient Boosting (XGBoost) | 99% |

---

## 🚀 المميزات

- **مقارنة بين 4 نماذج** — التبديل بينها في الوقت الفعلي
- **نسبة الثقة** — كل نموذج يعرض درجة الثقة بجانب التنبؤ
- **معالجة ذكية للبيانات** — القيم الصفرية في الحقول السريرية تُستبدل بالوسيط (median) لأن الصفر غير منطقي طبياً
- **واجهة نظيفة** — مبنية بـ Streamlit لتجربة سريعة وسهلة

## 📊 مجموعة البيانات

- **المصدر:** مجموعة بيانات PIMA Indians للسكري
- **الحجم:** 2,460 سجل مريض
- **الميزات:** 8 سمات سريرية
- **الهدف:** `Outcome` — 0 (لا يوجد سكري) / 1 (مصاب بالسكري)

### الميزات المُدخلة

| الميزة | الوصف |
|---|---|
| Pregnancies | عدد مرات الحمل |
| Glucose | تركيز الجلوكوز في البلازما |
| BloodPressure | ضغط الدم الانبساطي (mm Hg) |
| SkinThickness | سماكة ثنية الجلد (mm) |
| Insulin | الأنسولين في المصل بعد ساعتين (mu U/ml) |
| BMI | مؤشر كتلة الجسم (kg/m²) |
| DiabetesPedigreeFunction | درجة الخطر الوراثي للسكري |
| Age | العمر بالسنوات |

## ⚙️ المعالجة المسبقة للبيانات

القيم الصفرية في الأعمدة التالية غير منطقية طبياً، لذلك تُستبدل بـ **الوسيط (median)** المحسوب من بيانات التدريب:

- `Glucose`, `BloodPressure`, `SkinThickness`, `Insulin`, `BMI`
